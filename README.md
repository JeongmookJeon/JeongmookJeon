## 🚀 Projects & Verification Experience

### 📟 1. RISC-V Processor Architecture Design & Verification
- **Description:** [RISC_V_Single_Cycle](https://github.com/JeongmookJeon/RISC_V_Single_Cycle) 기반 프로세서 아키텍처 설계 및 검증 수행
- **Key Roles:**
  - Instruction Memory 및 ALU 모듈 설계
  - AXI 버스 인터페이스 통합 및 데이터 트랜잭션 검증

### 🧪 2. AXI-to-SPI Bridge Verification using UVM
- **Description:** AXI 기반 SPI Master/Slave 환경의 검증 체계 구축 (관련 레포: [MicroBlaze_SPI_I2C_UVM_Verification](https://github.com/JeongmookJeon/MicroBlaze_SPI_I2C_UVM_Verification))
- **Key Roles:**
  - UVM 프레임워크를 활용한 Testbench 환경 구성 (Agent, Scoreboard, Sequence 구현)
  - Datapath 검증 및 Verification Plan 수립

### ⚙️ 3. FIFO Module & Digital Logic Design
- **Description:** 데이터 버퍼링 및 동기화를 위한 FIFO 모듈 및 디지털 논리 회로 설계 ([SR04_DHT11_UART_FIFO_StopWatch_Watch](https://github.com/JeongmookJeon/SR04_DHT11_UART_FIFO_StopWatch_Watch) 연계)
- **Key Roles:**
  - SystemVerilog 및 Verilog를 활용한 RTL 구현
  - 클럭 도메인 교차(CDC) 이슈 고려 및 시뮬레이션 검증

### 🧠 4. On-Device AI & Vision Processing
- **[OnDevice_AI_YOLO_Ollama_AIVIS](https://github.com/JeongmookJeon/OnDevice_AI_YOLO_Ollama_AIVIS)**: YOLO 기반 객체 탐지와 Ollama LLM을 결합한 Edge AI 시스템 구축.
- **[OV7670_VGA_No_STM32](https://github.com/JeongmookJeon/OV7670_VGA_No_STM32)**: MCU 의존성 없이 순수 FPGA 로직만을 이용하여 OV7670 카메라 영상을 VGA로 출력하는 하드웨어 가속 처리 구현.
